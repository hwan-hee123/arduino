# HELIOS ☀ — 이족 보행 로봇

나만의 이족 보행 로봇 **HELIOS** 펌웨어.

## 구성 (기본값)
- 다리당 서보 3개: 엉덩이 / 무릎 / 발목 → 총 6개
- 보드: Arduino Uno / Nano (또는 PCA9685 서보 드라이버)

## 파일
| 파일 | 설명 |
|------|------|
| `helios.ino` | 메인 스케치 (셋업, 걸음걸이, 시리얼 명령) |
| `config.h`   | 핀 배치·중립각·방향·걸음 파라미터 (하드웨어에 맞게 수정) |

## 사용법
1. `config.h`에서 서보 핀과 중립 각도를 실제 조립에 맞게 보정
2. 아두이노에 업로드
3. 시리얼 모니터(9600 baud) 명령:
   - `n` : 중립 자세로 서기
   - `w` : 앞으로 걷기
   - `s` : 정지
