# HELIOS ☀ — 휴머노이드 로봇

나만의 휴머노이드 로봇 **HELIOS** 펌웨어. (PCA9685 16채널 서보)

## 하드웨어 구성
- 서보 드라이버: **PCA9685** (I2C, 주소 `0x40`)
- 서보 16개

### 채널 배선표
| 채널 | 관절 | 채널 | 관절 |
|------|------|------|------|
| 0 | 머리 | 8 | 오른쪽 고관절 |
| 1 | 허리 | 9 | 오른쪽 허벅지 |
| 2 | 오른쪽 어깨(상하) | 10 | 오른쪽 종아리 |
| 3 | 오른쪽 어깨(벌림) | 11 | 오른쪽 발목 |
| 4 | 오른쪽 팔꿈치 | 12 | 왼쪽 고관절 |
| 5 | 왼쪽 어깨(상하) | 13 | 왼쪽 허벅지 |
| 6 | 왼쪽 어깨(벌림) | 14 | 왼쪽 종아리 |
| 7 | 왼쪽 팔꿈치 | 15 | 왼쪽 발목 |

## 필요 라이브러리
- **Adafruit PWM Servo Driver Library** (아두이노 라이브러리 매니저에서 설치)

## 파일
| 파일 | 설명 |
|------|------|
| `helios.ino` | 메인 스케치 (셋업·걸음걸이·손흔들기·시리얼 명령) |
| `config.h`   | 채널·중립각·방향·걸음 파라미터 (하드웨어에 맞게 수정) |

## 사용법
1. `config.h`의 `NEUTRAL_ANGLES`를 조립 상태에서 로봇이 똑바로 서도록 보정
2. 좌우가 반대로 움직이면 `SERVO_DIRECTIONS`의 부호(+1/-1)를 조정
3. 아두이노에 업로드
4. 시리얼 모니터(9600 baud) 명령:
   - `n` : 중립(차렷) 자세
   - `w` : 앞으로 걷기
   - `a` : 손 흔들기 (인사)
   - `s` : 정지

> ⚠️ 첫 실행 전, 서보에 무리가 가지 않도록 다리를 살짝 들고 중립 자세부터 확인하세요.
